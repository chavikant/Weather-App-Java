# Weather-App-Java
Weather App

Overview

The Weather App is a simple and user-friendly Android application built using Java and a Weather API. This app provides real-time weather updates for any location, featuring a visually appealing UI with dynamic weather visuals that reflect current weather conditions.

Features

Real-Time Weather Updates: Fetches weather data from an external Weather API.

Basic UI: Displays the temperature and the weather of the perticular location.

Search Functionality: Allows users to search for weather updates of different locations.

Temperature & Conditions Display: Shows temperature and  humidity.

App Logo: Thinking of including one soon.

Technologies Used

Java (Android Development)

Android Studio (Development Environment)

Weather API (Fetches weather data)

XML (UI Design)

Installation

Clone this repository:

git clone https://github.com/your-username/weather-app.git

Open the project in Android Studio.

Obtain an API key from your preferred Weather API provider (e.g., OpenWeatherMap, WeatherAPI).

Add the API key to the strings.xml file:

<string name="weather_api_key">YOUR_API_KEY_HERE</string>

Run the project on an emulator or a physical Android device.

Usage

Launch the app.

Enter a city name in the search bar.

View real-time weather details with dynamic visuals.
